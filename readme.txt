# SumEdgeLengths Plugin

This is a Blender plugin that calculates the total length of all selected edges and displays the sum in the status bar.

## Installation

1. Open Blender and go to `Edit` > `Preferences`.
2. In the Preferences window, click on the `Add-ons` tab.
3. Click on the `Install...` button and select the downloaded plugin file.
4. Enable the plugin by checking the checkbox next to its name.

## Usage

1. Select an object and enter Edit mode.
2. Select multiple edges.
3. Right-click and choose "Sum Edge Lengths"
4. The total edge length will appear in Blender's status bar.
